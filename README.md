# 🌍 Tripster - Turismo e Curiosidades

O **Tripster** é um site de turismo criado para inspirar viajantes a conhecerem destinos incríveis pelo Brasil e pela América do Sul. Aqui você encontra descrições envolventes, dicas de viagem e os principais pontos turísticos de cada cidade, tudo em um só lugar.

O objetivo do projeto é unir informação e inspiração, ajudando o usuário a descobrir novos lugares e planejar suas próximas viagens de forma simples e agradável.

---

## ✨ Funcionalidades

- 📍 Navegação entre cidades turísticas
- 🏖️ Conteúdos detalhados sobre cada destino
- 🧭 Sidebar interativa para seleção de cidades
- 📖 Textos organizados com scroll para melhor leitura
- 🎨 Interface simples e focada na experiência do usuário

---

## 🗺️ Destinos disponíveis

No Tripster você pode explorar:

- 🇧🇷 Rio de Janeiro  
- 🌴 Fortaleza  
- 🏝️ Florianópolis  
- 🇦🇷 Buenos Aires  
- 🏔️ Bariloche  

Cada destino conta com uma seleção dos principais pontos turísticos, incluindo praias, centros históricos, atrações culturais, natureza e experiências locais.

---

## 🧩 Estrutura do projeto

O projeto foi desenvolvido em React e possui componentes organizados da seguinte forma:

- **Header** → Responsável pelo título e pela apresentação inicial do site  
- **Sidebar** → Menu lateral interativo para navegação entre as cidades  
- **Conteúdo dinâmico** → Exibição dos textos de acordo com a cidade selecionada, sendo cada destino estruturado como um componente próprio  

Essa organização facilita a manutenção do código e permite a expansão do projeto com novos destinos de forma simples e escalável.
